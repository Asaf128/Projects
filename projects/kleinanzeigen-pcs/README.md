# Kleinanzeigen-PCs

PC-Teile und Builds verwalten

## Beschreibung

Dieses Projekt ermöglicht die Verwaltung von PC-Teilen und Builds über ein Dashboard-Interface.

## Technologien

- React 18
- Vite
- Supabase (Backend)
- Vercel Analytics & Speed Insights

## Installation

```bash
npm install
```

## Entwicklung

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Dashboard für PC-Teile Verwaltung
- Build-Konfigurationen
- Datenbank-Integration mit Supabase
- Responsive Design
