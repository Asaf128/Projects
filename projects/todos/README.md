# To-Do's

Kanban Board für Task-Management

## Beschreibung

Dieses Projekt bietet ein Kanban-Board für die Verwaltung von Aufgaben mit Drag & Drop Funktionalität.

## Technologien

- React 18
- Vite
- Supabase (Backend)
- Vercel Analytics & Speed Insights

## Installation

```bash
npm install
```

## Entwicklung

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Kanban Board mit 5 Status-Spalten
- Drag & Drop Funktionalität
- Task-Verwaltung mit Prioritäten
- Benutzerzuweisung
- Fälligkeitsdaten
- Automatische Task-Nummerierung (TNR-1, TNR-2, etc.)
