# Beitragende

Vielen Dank für Ihr Interesse an diesem Projekt! Hier finden Sie alle Informationen, die Sie benötigen, um beizutragen.

## 🚀 Erste Schritte

### Voraussetzungen

- Node.js (Version 18 oder höher)
- npm oder yarn
- Git

### Einrichtung

1. Repository forken
2. Lokal klonen:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Projects.git
   ```
3. Abhängigkeiten installieren:
   ```bash
   npm install
   ```
4. Branch erstellen:
   ```bash
   git checkout -b feature/your-feature-name
   ```

## 📝 Entwicklungsrichtlinien

### Code-Style

- Verwenden Sie ESLint für die Code-Formatierung
- Halten Sie die Dateien kompakt und gut strukturiert
- Kommentieren Sie komplexe Logik

### Commit-Nachrichten

Format: `type(scope): description`

Beispiele:
- `feat: add new project tile`
- `fix: resolve login issue`
- `docs: update README`

### Branch-Namen

Format: `type/description`

Beispiele:
- `feature/add-dark-mode`
- `bugfix/fix-navigation`
- `docs/update-contributing`

## 🔄 Pull Request Prozess

1. Stellen Sie sicher, dass Ihr Code die ESLint-Regeln einhält
2. Aktualisieren Sie die Dokumentation, falls erforderlich
3. Erstellen Sie einen Pull Request mit einer klaren Beschreibung
4. Warten Sie auf die Überprüfung durch die Maintainer
